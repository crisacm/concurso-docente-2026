# Concurso Docente 2026

Aplicación web para simular el concurso docente 2026 bajo las normativas vigentes utilizando Inteligencia Artificial. Evaluando así la capacidad de aplicar la normativa en casos reales de la vida escolar.

La generación de preguntas y escenarios se hará utilizando de IA alimentada con la normativa vigente para el año 2026:

- Constitución Política: Especialmente el ingreso por mérito (Art. 125).
- Ley 115 de 1994: Los fines de la educación y el desarrollo integral.
- Decreto 1278 de 2002: El estatuto de profesionalización que regula el ingreso y ascenso.
- Resolución 3842 de 2022: El manual de funciones que define qué debe saber hacer cada docente.
- Guía 34 del MEN: Fundamental para preguntas sobre gestión institucional y planes de mejoramiento.
- Ley 1620 y Ruta de Atención Integral: Para casos de convivencia escolar y matoneo.
- Decreto 1421 de 2017: Para todo lo relacionado con educación inclusiva y discapacidad.
- Derechos Básicos de Aprendizaje (DBA): Para el componente de planificación y currículo
